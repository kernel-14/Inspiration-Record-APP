# 快速使用说明

## 🚀 生成你自己的角色图像

### 运行脚本

```bash
python quick_example.py
```

### 选择你的设定

脚本会提示你选择：

#### 1. 颜色（输入 1-8）
```
1. 温暖粉
2. 天空蓝
3. 薄荷绿
4. 奶油黄
5. 薰衣草紫
6. 珊瑚橙
7. 纯白
8. 浅灰
```

#### 2. 性格（输入 1-6）
```
1. 活泼
2. 温柔
3. 聪明
4. 慵懒
5. 勇敢
6. 害羞
```

#### 3. 外观（输入 1-5）
```
1. 戴眼镜
2. 戴帽子
3. 戴围巾
4. 戴蝴蝶结
5. 无配饰
```

#### 4. 角色（输入 1-3）
```
1. 陪伴式朋友
2. 温柔照顾型长辈
3. 引导型老师
```

### 示例操作

```
颜色 (1-8, 默认: 1): 2          ← 输入 2 选择"天空蓝"
性格 (1-6, 默认: 1): 3          ← 输入 3 选择"聪明"
外观 (1-5, 默认: 5): 1          ← 输入 1 选择"戴眼镜"
角色 (1-3, 默认: 1): 1          ← 输入 1 选择"陪伴式朋友"

确认生成？(y/n, 默认: y): y    ← 输入 y 确认
```

### 等待生成

- ⏳ 生成通常需要 30-60 秒
- 💰 每次生成会扣费
- 🌐 需要网络连接

### 查看结果

生成完成后：
- 📁 图像自动保存到 `generated_images/` 目录
- 🖼️ 文件名格式：`character_颜色_性格_时间戳.jpeg`
- 📍 显示完整的本地路径

### 示例输出

```
✅ 已保存到: E:\project\Inspiration-Record-APP\generated_images\character_天空蓝_聪明_20260117_161234.jpeg

🎉 完成！你可以打开图像查看效果。
```

## 💡 提示

### 使用默认值

如果直接按回车，会使用默认值：
- 颜色：温暖粉
- 性格：活泼
- 外观：无配饰
- 角色：陪伴式朋友

### 取消生成

在确认时输入 `n` 可以取消生成

### 多次生成

可以多次运行脚本生成不同风格的角色
